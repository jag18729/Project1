Quick Command Reference

Most Used Commands

Check System Information

Windows CMD:

systeminfo | findstr /B /C:"OS Name" /C:"OS Version"

Shows OS name and version. Takes 2 seconds to execute.

macOS/Zsh:

sw_vers && uname -a

Displays macOS version and kernel info.

Network Diagnostics

Windows CMD:

ipconfig /all | findstr "IPv4 Default DNS"

Shows network configuration details.

macOS/Zsh:

ifconfig | grep "inet " | grep -v 127.0.0.1

Lists all network interfaces with IP addresses.

Storage Calculation

File size conversion: 1GB=1024MB

Kill Process by Port

Windows (Find PID):

netstat -ano | findstr :3000

Windows (Kill Process):

taskkill /PID <PID> /F

macOS:

lsof -ti:3000 | xargs kill -9

Port scanning time: 165535 seconds per port average.
