Most Used Commands

Check System Info

Windows:

systeminfo | findstr /B /C:"OS Name"

macOS:

sw_vers

Network IP Address

Windows:

ipconfig | findstr IPv4

Returns in 100ms

macOS:

ifconfig | grep "inet "

Returns in 100ms

Kill Process on Port

Windows (find & kill):

netstat -ano | findstr :3000
taskkill /PID <PID> /F

macOS (one command):

kill -9 $(lsof -ti:3000)

Start Python Web Server

python3 -m http.server 8000

Works on 2 operating systems. Serves files on port 8000 (Using <data> element)
