Linux Commands

15 Essential Commands for System Management

How to Open Terminal on Linux

Method 1: Press Ctrl + Alt + T (works on most distributions)

Method 2: Right-click on desktop, select "Open Terminal"

Method 3: Search for "Terminal" in applications menu

Ubuntu: Click "Show Applications" and search "Terminal"

Top 15 Linux Commands

1. List Files and Directories

ls -la

Shows all files including hidden ones with permissions

2. Change Directory

cd /path/to/directory

Navigate to a specific directory

3. Print Working Directory

pwd

Shows your current location in the file system

4. Create Directory

mkdir newfolder

Creates a new directory

5. Remove Files

rm filename

Deletes a file (use -r for directories)

6. Copy Files

cp source destination

Copies files or directories (use -r for directories)

7. Move/Rename Files

mv oldname newname

Moves or renames files and directories

8. View File Contents

cat filename

Displays the entire file content

9. Search for Text

grep "pattern" filename

Searches for text patterns in files

10. Check Disk Usage

df -h

Shows disk space usage in human-readable format

11. View Running Processes

ps aux

Lists all running processes with details

12. Kill a Process

kill -9 PID

Forcefully terminates a process by its ID

13. Change File Permissions

chmod 755 filename

Sets read, write, and execute permissions

14. System Information

uname -a

Displays system and kernel information

15. Network Configuration

ifconfig

Shows network interface configuration

Distribution Package Managers

Ubuntu/Debian (APT)

sudo apt update

Updates package list

sudo apt install package-name

Installs a package

sudo apt remove package-name

Removes a package

Arch Linux (Pacman)

sudo pacman -Syu

Updates system

sudo pacman -S package-name

Installs a package

sudo pacman -R package-name

Removes a package

Red Hat/Fedora (YUM/DNF)

sudo dnf update

Updates system

sudo dnf install package-name

Installs a package

sudo dnf remove package-name

Removes a package

Quick Tips

Use Tab for auto-completion

Use Up/Down arrows to navigate command history

Use Ctrl+C to cancel a running command

Use Ctrl+L to clear the terminal

Use man command-name for help
