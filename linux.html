Linux System Administration

Practical Commands for Server & Network Management

How to Open Terminal on Linux

Method 1: Press Ctrl + Alt + T (works on most distributions)

Method 2: Right-click on desktop, select "Open Terminal"

Method 3: Search for "Terminal" in applications menu

Ubuntu: Click "Show Applications" and search "Terminal"

Web Server Management

Apache Web Server

sudo systemctl start apache2

Start Apache web server

sudo systemctl stop apache2

Stop Apache web server

sudo systemctl restart apache2

Restart Apache web server

sudo systemctl status apache2

Check Apache status

sudo systemctl enable apache2

Enable Apache to start on boot

Nginx Web Server

sudo systemctl start nginx

Start Nginx web server

sudo systemctl reload nginx

Reload Nginx configuration without dropping connections

sudo nginx -t

Test Nginx configuration for errors

Simple HTTP Server (Python)

python3 -m http.server 8000

Start a simple web server on port 8000

php -S localhost:8080

Start PHP built-in web server

Process & Service Management

Killing Processes

ps aux | grep process_name

Find process ID by name

kill PID

Terminate process gracefully

kill -9 PID

Force kill a process

killall process_name

Kill all processes with given name

pkill -f pattern

Kill processes matching pattern

Process Monitoring

top

Real-time process viewer (press q to quit)

htop

Interactive process viewer (if installed)

ps -ef

Show all running processes

lsof -i :8080

Find process using port 8080

netstat -tulpn

Show all listening ports and processes

Systemctl Service Management

Basic Service Control

sudo systemctl start service_name

Start a service

sudo systemctl stop service_name

Stop a service

sudo systemctl restart service_name

Restart a service

sudo systemctl reload service_name

Reload service configuration

sudo systemctl status service_name

Check service status

Service Configuration

sudo systemctl enable service_name

Enable service to start on boot

sudo systemctl disable service_name

Disable service from starting on boot

systemctl list-units --type=service

List all running services

systemctl list-units --failed

List failed services

journalctl -u service_name

View service logs

File Permissions & Ownership

Permission Management

ls -la

View files with permissions

chmod 755 filename

Owner: rwx, Group: r-x, Others: r-x

chmod 644 filename

Owner: rw-, Group: r--, Others: r--

chmod 600 filename

Owner: rw-, Group: ---, Others: ---

chmod +x script.sh

Make file executable

chmod -R 755 directory

Recursively set permissions

Ownership

chown user:group filename

Change file owner and group

chown -R user:group directory

Recursively change ownership

chgrp group filename

Change group only

Network Configuration & Testing

NetworkManager (nmcli)

nmcli device status

Show network devices status

nmcli connection show

List all connections

nmcli connection up connection_name

Activate a connection

nmcli connection down connection_name

Deactivate a connection

nmcli device wifi list

List available WiFi networks

nmcli device wifi connect SSID password PASSWORD

Connect to WiFi network

IP Commands

ip addr show

Show all IP addresses

ip a

Short version of ip addr show

ip route show

Show routing table

ip link set dev eth0 up

Bring network interface up

ip link set dev eth0 down

Bring network interface down

Legacy Network Commands

ifconfig

Display network interfaces (if installed)

ifconfig eth0 192.168.1.100

Set IP address

Network Testing

ping google.com

Test connectivity

traceroute google.com

Trace network path

nslookup domain.com

DNS lookup

dig domain.com

Detailed DNS information

ss -tulpn

Show listening ports (modern netstat)

curl -I https://example.com

Get HTTP headers

wget https://example.com/file.zip

Download file from web

Firewall Management

UFW (Ubuntu Firewall)

sudo ufw status

Check firewall status

sudo ufw enable

Enable firewall

sudo ufw allow 80

Allow port 80 (HTTP)

sudo ufw allow 443

Allow port 443 (HTTPS)

sudo ufw allow ssh

Allow SSH connections

sudo ufw deny 3306

Block MySQL port

iptables (Advanced)

sudo iptables -L

List all rules

sudo iptables -A INPUT -p tcp --dport 80 -j ACCEPT

Allow incoming HTTP

Quick Tips

Ctrl+C: Kill current process

Ctrl+Z: Suspend process (use 'fg' to resume)

Ctrl+L: Clear terminal

Tab: Auto-completion

!!: Repeat last command

sudo !!: Repeat last command with sudo

history: Show command history

man command: Show manual for command
