How to Open Terminal on macOS

Method 1: Press Command + Space, type "Terminal", press Enter

Method 2: Go to Applications → Utilities → Terminal

Method 3: Use Spotlight (Command + Space) and search "Terminal"

Pro Tip: Install iTerm2 for enhanced terminal features

Stop Apache web server

Nginx (via Homebrew)

Process & Task Management

Killing Processes

ps aux | grep process_name

Find process ID by name

kill PID

Terminate process gracefully

Force kill a process

killall process_name

Kill all processes with given name

pkill -f pattern

Kill processes matching pattern

killall Finder

Restart Finder

killall Dock

Restart Dock

File Permissions & Ownership

Permission Management

ls -la

View files with permissions

chmod 755 filename

Owner: rwx, Group: r-x, Others: r-x

chmod 644 filename

Owner: rw-, Group: r--, Others: r--

chmod 600 filename

Owner: rw-, Group: ---, Others: ---

chmod +x script.sh

Make file executable

chmod -R 755 directory

Recursively set permissions

Ownership

chown user:group filename

Change file owner and group

chown -R user:staff directory

Recursively change ownership

chgrp admin filename

Change group only

Access Control Lists

chmod +a "user allow read,write" file

Add ACL permission

chmod -a "user allow read,write" file

Remove ACL permission

ls -le

View ACL permissions

ifconfig

Network Utility

List all network hardware

networksetup -getinfo Wi-Fi

networksetup -setairportpower en0 off

networksetup -setairportpower en0 on

Turn WiFi on

airport -s

Scan for WiFi networks

pfctl (Packet Filter)
