Top 10 macOS Commands for IT Support

1. Reset WiFi ("WiFi stopped working!")

sudo ifconfig en0 down && sudo ifconfig en0 up

The macOS network reset - because turning WiFi off/on in menu bar "didn't work"

2. Force Quit Frozen Apps

killall Finder
killall Dock

When the spinning beach ball won't stop

3. Clear Application Cache

rm -rf ~/Library/Caches/*

When apps "act weird" - the macOS equivalent of clearing cookies

4. Fix Permissions ("Can't save files!")

sudo chown -R $(whoami) ~/Desktop ~/Documents

When mysterious permission errors appear

5. Reset SMC (Intel Macs)

sudo pmset -a hibernatemode 0

For weird hardware issues - tell them to also hold Shift-Control-Option-Power

6. Show Hidden Files

defaults write com.apple.finder AppleShowAllFiles -bool true && killall Finder

For finding those .DS_Store files everywhere

7. Disable Gatekeeper (Temporarily)

sudo spctl --master-disable
sudo spctl --master-enable

When "App can't be opened" - use with caution

8. Check System Integrity

diskutil verifyVolume /

The macOS health check

9. Clear DNS & Reboot mDNSResponder

sudo dscacheutil -flushcache && sudo killall -HUP mDNSResponder

When websites are "broken" but internet works

10. Emergency Safe Mode Boot

sudo nvram boot-args="-x"
sudo nvram -d boot-args

Set safe mode for next boot, and command to clear it
