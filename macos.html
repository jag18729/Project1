macOS Terminal Commands

15 Essential Commands for System Management

How to Open Terminal on macOS

Method 1: Press Command + Space, type "Terminal", press Enter

Method 2: Go to Applications → Utilities → Terminal

Method 3: Use Spotlight (Command + Space) and search "Terminal"

Top 15 macOS Commands

1. List Files and Directories

ls -la

Shows all files including hidden ones

2. Change Directory

cd ~/Desktop

Navigate to a specific directory

3. Print Working Directory

pwd

Shows current directory path

4. Create Directory

mkdir newfolder

Creates a new folder

5. Remove Files

rm filename.txt

Deletes a file (use -r for folders)

6. Copy Files

cp source.txt destination.txt

Copies files to another location

7. Move/Rename Files

mv oldname.txt newname.txt

Moves or renames files

8. View File Contents

cat filename.txt

Displays file contents

9. Clear Terminal

clear

Clears the terminal screen

10. Show Network Info

ifconfig

Displays network configuration

11. Ping Network

ping google.com

Tests network connectivity

12. View Running Processes

ps aux

Lists all running processes

13. Kill Process

kill -9 PID

Force stops a process by ID

14. System Information

sw_vers

Shows macOS version info

15. Disk Usage

df -h

Shows disk space usage

macOS Specific Commands

Open Files/Apps

open filename.txt

Opens file with default application

Show Hidden Files

defaults write com.apple.finder AppleShowAllFiles YES

Makes hidden files visible in Finder

Say Text Aloud

say "Hello World"

macOS speaks the text

Screenshot

screencapture screenshot.png

Takes a screenshot

Restart Finder

killall Finder

Restarts Finder application

Quick Tips

Use Tab for auto-completion

Use Up/Down arrows for command history

Use Ctrl+C to stop a running command

Use Cmd+K to clear terminal

Use man command-name for help
