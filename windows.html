Windows System Administration

Practical Commands for Network & Process Management

Performance Metrics: 1100 second response time

How to Open Command Prompt on Windows

Method 1: Press Windows + R, type "cmd", press Enter

Method 2: Search "cmd" or "Command Prompt" in Start Menu

Method 3: Right-click Start button, select "Windows Terminal" or "Command Prompt"

Pro Tip: Add "Run as Administrator" for elevated privileges

Network Configuration & Testing

IP Configuration

ipconfig

Show basic network configuration

ipconfig /all

Show detailed network configuration including MAC addresses

ipconfig /release

Release current IP address

ipconfig /renew

Renew IP address from DHCP

ipconfig /flushdns

Clear DNS resolver cache

ipconfig /displaydns

Display DNS resolver cache contents

Network Testing

ping 8.8.8.8

Test connectivity to Google DNS

ping -t google.com

Continuous ping (Ctrl+C to stop)

ping -n 10 192.168.1.1

Ping 10 times to specific IP

tracert google.com

Trace route to destination

pathping google.com

Combination of ping and tracert with statistics

nslookup google.com

DNS lookup for domain

nslookup google.com 8.8.8.8

DNS lookup using specific DNS server

Network Connections

netstat -an

Show all network connections and ports

netstat -anob

Show connections with process IDs (requires admin)

netstat -r

Display routing table

arp -a

Show ARP cache (IP to MAC mappings)

route print

Display routing table

netsh wlan show profiles

Show saved WiFi profiles

netsh wlan show profile name="WiFiName" key=clear

Show WiFi password (requires admin)

Process & Task Management

View Processes

tasklist

List all running processes

tasklist /FI "MEMUSAGE gt 100000"

Show processes using more than 100MB RAM

tasklist /FI "STATUS eq NOT RESPONDING"

Show hung processes

tasklist /SVC

Show services in each process

tasklist | findstr chrome

Find specific process

Kill Processes

taskkill /PID 1234

Kill process by Process ID

taskkill /PID 1234 /F

Force kill process

taskkill /IM notepad.exe

Kill process by name

taskkill /IM chrome.exe /F

Force kill all Chrome processes

taskkill /T /F /PID 1234

Kill process tree (process and children)

Performance Monitoring

wmic process list brief

List processes with basic info

wmic process where "name='chrome.exe'" get processid

Get PID for specific process

typeperf "\Processor(_Total)\% Processor Time" -sc 10

Monitor CPU usage for 10 samples

Windows Service Control

Service Commands (SC)

sc query

List all services

sc query state=all

List all services including inactive

sc query "Windows Update"

Check specific service status

sc start "ServiceName"

Start a service

sc stop "ServiceName"

Stop a service

sc config "ServiceName" start=auto

Set service to start automatically

sc config "ServiceName" start=disabled

Disable service

NET Commands

net start

List running services

net start "Windows Defender"

Start a service

net stop "Print Spooler"

Stop a service

net user

List user accounts

net user username /active:yes

Enable user account

net localgroup administrators

Show administrators group members

iisreset

iisreset /stop

iisreset /start

iisreset /status

Application Pool Management

appcmd list apppool

List all application pools

appcmd start apppool /apppool.name:"DefaultAppPool"

Start application pool

appcmd stop apppool /apppool.name:"DefaultAppPool"

Stop application pool

appcmd recycle apppool /apppool.name:"DefaultAppPool"

Recycle application pool

File Permissions & Security

File Permissions (ICACLS)

icacls filename.txt

View file permissions

icacls filename.txt /grant Users:F

Grant full control to Users group

icacls filename.txt /grant Username:R

Grant read permission to specific user

icacls filename.txt /deny Username:W

Deny write permission

icacls folder /T /grant Users:F

Recursively grant permissions

icacls * /reset /T

Reset permissions to inherited

File Attributes

attrib +h filename.txt

Set file as hidden

attrib +r filename.txt

Set file as read-only

attrib -h -r filename.txt

Remove hidden and read-only attributes

cipher /e /s:folder

Encrypt folder and contents

System Information & Diagnostics

System Info

systeminfo

Detailed system configuration

systeminfo | findstr /B "OS Name"

Show OS name only

wmic os get caption,version

Show Windows version

wmic cpu get name,numberofcores

Show CPU information

wmic memorychip get capacity

Show RAM capacity

Disk Management

wmic logicaldisk get size,freespace,caption

Show disk space

fsutil fsinfo drives

List all drives

chkdsk C: /f

Check disk for errors and fix

defrag C: /A

Analyze disk fragmentation

sfc /scannow

System file checker (requires admin)
