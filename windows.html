Windows CMD Commands

15 Essential Commands for System Management

How to Open Command Prompt on Windows

Method 1: Press Windows + R, type "cmd", press Enter

Method 2: Search "cmd" or "Command Prompt" in Start Menu

Method 3: Right-click Start button, select "Windows Terminal" or "Command Prompt"

Pro Tip: Add "Run as Administrator" for elevated privileges

Top 15 Windows Commands

1. List Directory Contents

dir

Shows files and folders in current directory

2. Change Directory

cd C:\Users\Username

Navigate to a specific directory

3. Clear Screen

cls

Clears the command prompt window

4. Create Directory

mkdir newfolder

Creates a new directory

5. Delete Files

del filename.txt

Deletes specified file

6. Copy Files

copy source.txt destination.txt

Copies files to another location

7. Move Files

move oldfile.txt C:\newfolder\

Moves files to another location

8. Rename Files

ren oldname.txt newname.txt

Renames a file or folder

9. View File Contents

type filename.txt

Displays the contents of a text file

10. IP Configuration

ipconfig

Shows network configuration

11. Ping Network

ping google.com

Tests network connectivity

12. List Running Tasks

tasklist

Shows all running processes

13. Kill Process

taskkill /PID 1234 /F

Terminates a process by its ID

14. System Information

systeminfo

Displays detailed system configuration

15. Check Disk

chkdsk C:

Checks disk for errors

Common Network Commands

Flush DNS Cache

ipconfig /flushdns

Clears DNS resolver cache

Show Network Stats

netstat -an

Displays all network connections

Trace Route

tracert google.com

Shows path to destination

ARP Table

arp -a

Shows IP to MAC address mappings

Network Share

net use

Shows mapped network drives

Quick Tips

Use Tab for auto-completion

Use arrow keys for command history

Use Ctrl+C to stop a running command

Use /? after any command for help

Run as Administrator for system commands
