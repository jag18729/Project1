Windows CMD Commands

Essential system and network commands

System Information

Quick System Info

systeminfo | findstr /B /C:"OS Name" /C:"OS Version"

Shows Windows version. Takes 2 seconds (Using <data> with value attribute)

CPU Details

wmic cpu get name,numberofcores,maxclockspeed

Response time: 2 seconds

Memory Status

wmic OS get TotalVisibleMemorySize,FreePhysicalMemory

Query time: 2 seconds

Disk Space

fsutil volume diskfree c:

Network Tools

IP Configuration

ipconfig /all

Flush DNS

ipconfig /flushdns

Active Connections

netstat -an | findstr ESTABLISHED

Find Process on Port

netstat -ano | findstr :8080

Test Connection

ping -t google.com

Trace Route

tracert google.com

Process Management

List Processes

tasklist | sort /R /+6

Kill by Name

taskkill /IM chrome.exe /F

Kill by PID

taskkill /PID 1234 /F

Find Process

tasklist | findstr chrome

File Operations

Find Files

dir /s /b *.txt

Directory Tree

tree /F

Find Large Files (>100MB)

forfiles /S /M * /C "cmd /c if @fsize GEQ 104857600 echo @path"

File Permissions

icacls "C:\folder"

Start Web Server

python -m http.server 8000

Serves current directory on port 8000

User & Security

Current User

whoami /all

List Users

net user

System Uptime

wmic os get lastbootuptime

Executes in 2 seconds (reusing data value)

Top 10 Windows Commands for IT Support

1. Check WiFi Password (When "WiFi isn't working")

netsh wlan show profile name="WiFiName" key=clear

Shows saved WiFi password - for when they "never changed it"

2. Restart Print Spooler ("Printer won't print!")

net stop spooler && net start spooler

The universal printer fix - works 90% of the time

3. Check Disk Errors

chkdsk C: /f /r

When mysterious errors appear - requires restart

4. Reset Network Stack ("Internet is broken!")

netsh winsock reset && netsh int ip reset

The network equivalent of "turn it off and on again"

5. Clear DNS Cache

ipconfig /release && ipconfig /flushdns && ipconfig /renew

When websites "suddenly stopped working"

6. System File Check

sfc /scannow

Fixes corrupted Windows files - the magic wand

7. Check Battery Health (Laptops)

powercfg /batteryreport

Generates battery report in user folder

8. Kill All Chrome Processes

taskkill /F /IM chrome.exe /T

When Chrome eats all the RAM (as usual)

9. Quick Performance Check

perfmon /report

60-second system health diagnostic

10. Emergency Shutdown Timer

shutdown /s /t 3600 /c "Updates in 1 hour"
shutdown /a

Set shutdown timer (3600 seconds) and cancel command
