IT Support Flowchart

  1. Is it plugged in? → Plug it in
  2. Is it turned on? → Turn it on
  3. Have you restarted it? → Restart it
  4. Is the cable connected? → Check both ends
  5. Did you Google the error? → Copy/paste to Stack Overflow
  6. Clear the cache → It's always the cache
  7. Check DNS → It's always DNS
  8. Update drivers → Or roll them back
  9. Run as Administrator → With great power...
  10. Call IT → "It was working yesterday"
