Windows CMD Commands

System diagnostics, network tools, and utilities for Windows

Windows Command Prompt Reference

Copy and paste these commands directly into CMD or PowerShell. Last tested on January 15, 2024 on Windows 11.

System Information & Diagnostics

Complete System Information

systeminfo

Displays detailed configuration about computer and OS. Runtime: 3-5 seconds

Quick System Summary

wmic computersystem get model,manufacturer,name,totalphysicalmemory

Shows computer model and RAM in bytes.

CPU Information

wmic cpu get name,maxclockspeed,numberofcores,numberoflogicalprocessors

Displays processor details. Clock speed in MHz.

Check Windows Version

winver

Opens dialog with Windows version. Alternative command-line:

ver && wmic os get caption,version,buildnumber

Storage Information

wmic diskdrive get model,size,status

Lists all disk drives. Size in bytes: bytes/109=GB

Check Free Disk Space

fsutil volume diskfree c:

Shows free space on C: drive in bytes.

Directory Size (PowerShell)

Get-ChildItem -Path C:\Users -Recurse | Measure-Object -Property Length -Sum

Calculates folder size recursively.

Network Diagnostics & Configuration

Network Configuration

ipconfig /all

Complete network adapter information.

DNS Cache

ipconfig /displaydns

Shows DNS resolver cache. Typical cache entries: 200+ domains

Flush DNS Cache

ipconfig /flushdns

Clears DNS cache. Execution time: 100ms

Network Statistics

netstat -an | findstr ESTABLISHED

Shows active connections.

Port Usage by Process

netstat -anob

Lists ports with process names (requires admin).

Test Network Speed to Server

ping -t -l 1000 google.com

Continuous ping with 1000 byte packets.

Trace Network Route

tracert -d google.com

Shows network path. Max hops: 30

ARP Table

arp -a

Displays IP to MAC address mappings.

Process & Service Management

List All Processes

tasklist /FO TABLE

Shows running processes with memory usage.

Find Process by Name

tasklist | findstr chrome

Filters processes containing "chrome".

Kill Process by PID

taskkill /PID 1234 /F

Force terminates process ID 1234.

Kill Process by Name

taskkill /IM chrome.exe /F

Terminates all Chrome instances.

List Windows Services

sc query state=all | findstr SERVICE_NAME

Shows all services. Typically 300+ services on Windows 11.

Check Service Status

sc query "Windows Update"

Shows specific service status.

Security & User Information

Current User Info

whoami /all

Displays user, groups, and privileges.

List All Users

net user

Shows all local user accounts.

User Account Details

net user %USERNAME%

Shows current user details including last password change.

Check Windows Firewall

netsh advfirewall show allprofiles

Displays firewall status for all profiles.

List Firewall Rules

netsh advfirewall firewall show rule name=all

Shows all firewall rules. Usually 500+ rules.

Performance Monitoring

Memory Usage Statistics

wmic OS get TotalVisibleMemorySize,FreePhysicalMemory

Shows RAM in KB. Formula: KB1024=MB

Top Memory Consuming Processes

tasklist /FO TABLE | sort /R /+6

Sorts processes by memory usage.

System Uptime

wmic os get lastbootuptime

Shows last boot time in format: YYYYMMDDHHmmss

Check System Files

sfc /scannow

Scans system files for corruption. Duration: 10-15 minutes

File & Directory Operations

Find Files by Extension

dir /s /b *.log

Recursively finds all .log files.

Find Large Files (>100MB)

forfiles /S /M * /C "cmd /c if @fsize GEQ 104857600 echo @path @fsize"

Lists files larger than 100 MB.

Directory Tree

tree /F /A

Shows folder structure with files using ASCII.

File Permissions

icacls "C:\Users\%USERNAME%\Documents"

Shows NTFS permissions for folder.

Create Symbolic Link

mklink /D "C:\LinkName" "C:\Target\Folder"

Creates directory symbolic link (requires admin).

PowerShell Quick Commands

Launch Python web server:

python -m http.server 8000

Or with Python 2:

python -m SimpleHTTPServer 8000

Server starts in 500ms on port 8000

Advanced System Diagnostics

Generate System Health Report

perfmon /report

Creates detailed 60-second performance report.

Event Log Errors (Last 24h)

wevtutil qe System /f:text /c:20 /q:"*[System[(Level=2) and TimeCreated[timediff(@SystemTime) <= 86400000]]]"

Shows recent system errors.

Driver Information

driverquery /FO LIST /V

Lists all installed drivers with details.

Battery Report (Laptops)

powercfg /batteryreport

Generates battery health HTML report.

Network Adapter Power

powercfg /devicequery wake_armed

Shows devices that can wake computer.
